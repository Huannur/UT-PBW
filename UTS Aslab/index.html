About Me

Halo, saya adalah M. Huannur, seorang mahasiswa dan pemilik website ini. Saya sangat menyukai hal-hal yang berkaitan dengan teknologi informasi, serta memiliki ketertarikan dalam hal analisis. Saya memulai website ini sebagai sarana untuk memenuhi Ujian Tengah Semester dari Mata Kuliah Pemrograman Berbasis Web. Melalui website ini, saya berharap dapat memberikan manfaat dan inspirasi bagi pembaca. Selain itu, saya juga memiliki pengalaman dalam organisasi yang terdapat dalam perkuliahan saat ini. Saya sangat senang bisa terhubung dengan Anda dan membantu Anda dalam hal apa pun yang saya bisa. Terima kasih telah berkunjung ke website saya dan jangan ragu untuk menghubungi saya jika Anda memiliki pertanyaan atau komentar.
